# ingresar a psql.exe
```
Cerrar conexión 	\q
Cambiar de Base de datos 	\c __base_datos__
Listar Bases de datos 	\l
Ver Definiciones 	\d __table__
Listar Schemas 	\dn
Listar funciones 	\df
Listar Vistas 	\dv
Ver código SLQ de la función 	\df+ __function
Pretty-format 	\x


```

```
psql -u postgres -W -h localhost
